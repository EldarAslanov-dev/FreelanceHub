# 🚀 FreelanceHub

> Azerbaijani freelance marketplace platform connecting clients with skilled professionals.

## 📋 About

FreelanceHub is a full-stack web application built as a graduation project at Code Academy Azerbaijan. It aims to fill the gap in the local freelance market by providing a dedicated platform for Azerbaijani freelancers and clients.

## ✨ Features

- 👤 User registration & authentication (JWT)
- 📝 Job posting & management
- 💼 Freelancer profiles & portfolios
- 🤝 Bidding system
- 📊 Admin dashboard
- 🤖 AI-assisted features (CV analysis, code suggestions)

## 🛠️ Tech Stack

- **Backend:** C#, ASP.NET Core
- **Database:** SQL Server, Entity Framework Core
- **DevOps:** Docker, GitHub Actions, CI/CD
- **AI Tools:** GitHub Copilot, ChatGPT API
